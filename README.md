<h1 align="center"> Falletter-Mobile </h1>

<p align="center">
  <b>오직 DSM 학생들만을 위한 익명 소통과 맞춤형 연결 서비스</b><br/>
  학교 커뮤니티 내 숨겨진 상호작용을 활성화하고,<br/>
  학생들 간의 자연스러운 교류를 돕는 <b>익명 편지 기반 커뮤니티 플랫폼</b>입니다.
</p>

<hr>

<h1 align="center"> 서비스 소개 </h1>

<p align="center">
단순한 메시지 전달을 넘어, 학생들이 자유롭게 의견을 나누고 새로운 관계를 형성할 수 있도록 돕는 플랫폼입니다.

<p align="center"> 💌 <b> 익명 편지 시스템 </b> — 솔직한 마음을 전달할 수 있는 공간</p>
<p align="center"> 🎯 <b> 맞춤형 연결 서비스 </b> — 공감과 취향 기반의 연결</p>
<p align="center"> 💬 <b> 커뮤니티 활성화 </b> — 학교 내 자연스러운 소통 촉진</p>
<p align="center"> 🔒 <b> DSM 기반 인증 </b> — 재학생만 이용 가능한 안전한 환경</p> </p>
